## Design Patterns Applied

### Container Widths
- **Forms (Simple)**: `max-w-2xl` - Join Vendor, Planning steps
- **Forms (Complex)**: `max-w-3xl` - Add Service, Edit Profile
- **Dashboards**: Full-width - My Wedding, Bookings, Guest List, Seating Planner
- **Detail Pages**: `max-w-4xl` or `max-w-5xl` - Product/service details

### Grid Patterns
- **Mobile**: 1-2 columns
- **Tablet (lg)**: 2-3 columns
- **Desktop (xl)**: 3-4 columns
- **Dashboard**: Special layouts (2/3 + 1/3 for Seating Planner)

### Responsive Spacing
- **Headers**: `p-4 lg:p-6`
- **Main Content**: `px-4 lg:px-6 py-6 lg:py-8`
- **Cards**: `p-4 lg:p-5`
- **Gaps**: `gap-3 lg:gap-4` or `gap-4 lg:gap-6`

### Text Sizing
- **Page Titles**: `text-lg lg:text-xl`
- **Section Headers**: `text-2xl lg:text-3xl`
- **Body Text**: `text-sm lg:text-base`
- **Stats**: `text-2xl lg:text-3xl` or `text-3xl lg:text-4xl`

### Button Sizing
- **Primary Actions**: `h-14 lg:h-16 text-base lg:text-lg`
- **Secondary Actions**: `h-10 lg:h-12 text-sm lg:text-base`
